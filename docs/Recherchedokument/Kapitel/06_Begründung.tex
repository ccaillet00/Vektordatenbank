\section{Begründung der Produktauswahl}
Für die praktische Demonstration wurde pgvector gewählt, da es den evolutionären Ansatz der Vektorsuche repräsentiert: Die Integration von KI-Funktionen in bewährte relationale Datenbanksysteme. Während spezialisierte Datenbanken oft eine isolierte Lösung darstellen, erlaubt pgvector die hybride Nutzung von strukturierten Geschäftsdaten und unstrukturierten Vektor-Embeddings innerhalb einer ACID-konformen Umgebung. Dies spiegelt eine häufige Anforderung in Unternehmen wider: Effizienz durch Nutzung bestehender Technologie-Stacks. \footcite{Gemini2026}
